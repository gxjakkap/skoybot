# สก๊อยบ์ บอธ

บอธง่๊ยๆธิ๊ฆ์จ่เปลิ๊่ญนฆ๊ฮคว๊ฒคุ๊ลเป็ณภ๊ษ๊สก๊อย โฎญใช๊ [skoy.js](https://www.npmjs.com/package/skoy)

## ฟิ๊เ๗อร์

เปลิ๊่ญนฆ๊อคว๊มภ๊๊๊ไธญเป็ณภ๊ษ๊สก๊อป์ย
 

## ษิ่งธิ๊ฆ์จำเป็น

* NodeJs 16

## ก๊รฏิฎฏั๊ง

### 1. ลงเเพ็กเกจธิ๊ฆ์ฏ๊ฮงใช๊

```bash
$ yarn install
```

### 2. กรอกโธเค่ฯเเล๊พริ๊ฟิกซ์

```js
//config.js
const token = ""
const prefix = ""

module.exports = {
    token,
    prefix
}

```

### 3. รัห์ลบฮธ

```bash
$ node .
```
